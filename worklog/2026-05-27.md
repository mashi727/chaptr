### TL;DR

- チャプター冒頭に `--` を付けても波形に斜線（除外ハッチング）が出ない件を調査 → **同一時刻 `0:00:00` に2チャプターが衝突**し区間幅0で描画されないのが原因と特定。
- 区間計算を GUI 非依存の純粋関数 `models.compute_excluded_regions()` に集約し、波形表示・単一ソースのエクスポートカット・テストを統一。

### このセッションでやったこと

- ユーザー報告「`--` を付けても斜線が出ない。OKなものとそうでないものがある」を起点に、斜線描画箇所を `grep` で特定（`chaptr/ui/widgets/waveform.py` の除外ハッチング）。

- 保存ロジック（`chapter_manager.py:411-414`）はリスト順書き出しのみで重複行を自動付加しないことを確認 → 重複は実データ由来。
- 修正方針として A（データ修正）/ B（コード恒久修正）を提示 → ユーザーが **B（波形表示の修正）** を選択。
- `_get_excluded_regions` を「時刻が厳密に大きい次チャプターまで」に変更（同時刻を飛ばす）。
- 既存テスト `test_waveform_regions.py` が**本番ロジックをコピー再現**していて本番バグを検知できない構造だったため、純粋関数 `compute_excluded_regions()` を `models.py` に切り出し、波形・テストが共有する形に refactor。回帰テスト2件追加。
- ユーザーが「お願いします」でエクスポート側整合＋コミットを承認 → エクスポートの除外計算も同関数へ統一。

### 重要な決定と理由

- **純粋関数の置き場所を `models.py` に** → GUI非依存で `ChapterInfo` と同居でき、波形（widget）・エクスポート（worker）・テストの3者が循環依存なく共有できる Single Source of Truth にするため。当初 `waveform.py` に置いたが、worker から widget を import するのは不自然なため models へ移動。
- **「次チャプター」を時刻が厳密に大きい最初のチャプターに変更** → 同時刻の重複（`0:00:00` に複数）で区間幅0／逆順区間が生じるのを防ぐ。整形済み（時刻順・重複なし）プロジェクトでは従来結果と完全一致し、**出力不変**であることを確認。
- **エクスポートは単一ソース経路 `export.py` のみ修正** → ユーザーのケースは単一ソース。複数ソースは `base.py::calculate_extraction_plan`（ソース内ローカル時間の別ロジック）、分割は `export_split.py`（既に `end_ms > start_ms` でゼロ幅を弾く）で、今回のケースに不要かつ出力リスクを避けるため未変更。
- **テストを本番関数呼び出しへ変更** → コピー再現だったため本番バグを検知できなかった構造的原因を解消（テストが本番コードを実際に検証するように）。
- **`main` ではなく作業ブランチにコミット** → デフォルトブランチへの直接コミットを避けるガードレールに従った。fast-forward マージ可能。
- **デッドコード `workers.py` への編集を revert** → パッケージに隠蔽され実行されない127KBの旧モノリス。混乱を避けるため変更を残さない判断。

- **現象**：`ImportError: cannot import name 'compute_excluded_regions' from 'chaptr.ui.workers' (/.../chaptr/ui/workers/__init__.py)`
- **原因**：同階層に `chaptr/ui/workers.py`（モジュール）と `chaptr/ui/workers/`（パッケージ）が**両方存在**し、Pythonはパッケージを優先。`workers.py` はアプリから一度も import されないデッドコードで、最初にそこを編集していた。
- **解決**：実体である `chaptr/ui/workers/export.py` の `ExportWorker._process_excluded_chapters` を修正し、`workers.py` の編集は `git checkout -- chaptr/ui/workers.py` で revert。

修正後の区間計算ロジック（`models.compute_excluded_regions` 抜粋）：

```python
sorted_chapters = sorted(chapters, key=lambda c: c.time_ms)
for i, ch in enumerate(sorted_chapters):
    if ch.title.startswith("--"):
        start_ms = ch.time_ms
        end_ms = duration_ms
        for nxt in sorted_chapters[i + 1:]:
            if nxt.time_ms > start_ms:      # 同時刻を飛ばし、厳密に大きい次まで
                end_ms = nxt.time_ms
                break
        excluded_regions.append((start_ms, end_ms))
```
